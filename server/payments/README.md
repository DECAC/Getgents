# Paiements — Mollie

**Hors scope total — aucun flux de paiement réel n'existe.**

Getgents ne déclenche jamais de paiement et ne stocke aucun moyen de paiement.
Pour les comptes tiers (ex. Booking.com), le paiement se fait entièrement hors Getgents, chez le prestataire.

Ce dossier matérialise la cible architecture sans rien implémenter.
